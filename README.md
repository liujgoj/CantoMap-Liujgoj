# CantoMap-Liujgoj

This repository contains a transliterated version of the **CantoMap** corpus, strictly converted into the **Liujgoj (溜歌粵語)** standalone Romanization/writing system.

## Dataset Features
- Strict syllable-by-syllable mapping based on the Liujgoj inventory.
- Preserved all original word boundaries (word-segmentation) and colloquial markers (`#`, `|`).
- Highly valuable for training Speech-Native Multimodal AI in Cantonese.
